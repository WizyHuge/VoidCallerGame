# VOID CALLER

**VOID CALLER** — это атмосферная 2D игра-головоломка с видом сверху, где главным инструментом выживания является звук. Вы находитесь в полной темноте и должны использовать эхолокацию, чтобы видеть окружение, решать загадки и найти выход.

## Особенности игры

*   **Уникальная механика зрения:** Вы видите мир только через звуковые волны.
*   **Процедурная генерация:** Карта подземелья создается заново при каждом запуске.
*   **Головоломки:** Для включения света нужно решать мини-игры (последовательность, удержание).

## Установка и запуск

**Скачайте и запустите последнюю версию игры в releases**

## Управление

| Клавиша | Действие |
|:---:|:---|
| **W, A, S, D** | Перемещение персонажа |
| **Мышь** | Прицеливание |
| **ЛКМ (Левая кнопка)** | Эхолокация |
| **E** | Взаимодействие |
| **1, 2, 3, 4** | Ввод кода (для пазлов с последовательностью) |
| **Пробел (Space)** | Удержание (для пазлов с удержанием) |
| **R** | Перезапуск игры |
| **ESC** | Выход |

## Цель игры

1.  Вы появляетесь в темном лабиринте.
2.  Используйте ЛКМ, чтобы сканировать местность и видеть стены.
3.  Найдите Генераторы.
4.  Подойдите к генератору и нажмите E, чтобы начать запуск.
5.  Решите головоломку генератора, чтобы включить свет.
6.  Активируйте все генераторы на уровне.
7.  Найдите Выходную дверь и покиньте уровень.

## Структура проекта

*   `main.py` — Точка входа в игру.
*   `game_window.py` — Основная логика игрового цикла и отрисовки.
*   `menu.py` — Главное меню.
*   `player.py` — Логика персонажа и движения.
*   `map.py` — Генерация карты.
*   `vision_system.py` — Отрисовка видимости.
*   `echolocation.py` — Логика звуковых волн и частиц.
*   `puzzle_system.py` — Логика мини-игр для генераторов.
*   `generator.py` / `exit_door.py` — Игровые объекты.
*   `core/constants.py` — Настройки баланса и константы.
